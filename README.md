# UIKits2 

A complete UIs for mobile and tablet, which include 16 categories.

 * [Start](#start-screen)
 * SignUp & Login
 * Walkthrough
 * Loading
 * Profiles
 * Feed
 * Article
 * Activity
 * Create
 * Chat
 * Contact
 * Store & Shop
 * Shopping Carts & Checkout
 * Navigation
 * Photos
 * Dashboard
